# Демонстрація частотних фільтрів — Streamlit застосунок

Інтерактивний застосунок на основі **Streamlit** для демонстрації фільтрації зображень у частотній області.  
Дозволяє завантажити зображення, виконати перетворення Фур’є, застосувати різні частотні фільтри, переглянути спектр та порівняти результати.

---

Додаток розгорнуто на платформі Streamlit Community Cloud. Доступ можна отримати за посиланням: https://filters-in-frequency-domain.streamlit.app/

---

## ✨ Можливості

- ✔️ Підтримка RGB та градацій сірого  
- ✔️ Обчислення та візуалізація спектру (логарифмічна шкала)  
- ✔️ Підтримувані фільтри:
  - Ідеальний зріз
  - Гаусовий
  - Баттерворта (із заданням порядку)
  - Для наведених вище низькочастотний та високочастотний варіанти і смугові фільтри (2 варіанти)
  - Фільтр Лапласа
  - Лаплас від Гауса (LoG)
- ✔️ Інтерактивна панель налаштувань:  
  значення зрізу, порядок, ширина смуги
- ✔️ Система підказок, що читається з JSON-файлу  
- ✔️ Порівняння результатів:
  - оригіналу   
  - відфільтрованого  
  - відображення спектру для обох варіантів
